Работа с Visual Studio Code

Теперь, когда VS Code установлен, давайте научимся им пользоваться и настроим для удобной работы с HTML и CSS.

Интерфейс VS Code

После запуска VS Code вы увидите чистый интерфейс с боковой панелью слева, основной рабочей областью и статусной строкой внизу.

Основные элементы интерфейса:

  • Activity Bar (слева) - иконки для переключения между различными панелями
  • Side Bar - показывает содержимое папки или другие панели в зависимости от выбранной иконки
  • Editor - основная область для редактирования файлов
  • Panel (внизу) - терминал, проблемы, вывод отладки и т.д.
  • Status Bar (внизу) - информация о текущем файле, кодировка, ветка Git и т.д.
Открытие папки с проектом

Для начала работы:

  1. Создайте папку для вашего проекта (например, "Мой сайт")
  2. В VS Code выберите File > Open Folder (Файл > Открыть папку)
  3. Выберите созданную папку

Теперь все файлы, которые вы будете создавать в этой папке, будут отображаться в боковой панели.

Создание первого HTML-файла
  1. В боковой панели нажмите на иконку "New File" (или правой кнопкой в панели Explorer > New File)
  2. Назовите файл "index.html"
  3. Начните вводить "html" и нажмите Tab, когда появится подсказка "html:5" - VS Code автоматически создаст базовую структуру HTML-документа
Полезные расширения

VS Code поддерживает расширения, которые делают работу удобнее. Для HTML/CSS разработки рекомендую установить:

  • Live Server - запускает локальный сервер с автоматической перезагрузкой страницы при изменениях
  • Prettier - автоматически форматирует код
  • HTML CSS Support - автодополнение для HTML и CSS
  • Auto Rename Tag - автоматически переименовывает парные HTML-теги

Для установки расширений:

  1. Нажмите на иконку расширений в Activity Bar
  2. Введите название расширения в поиск
  3. Нажмите "Install"
Совет: Используйте сочетания клавиш для ускорения работы:
  • Ctrl + N - новый файл
  • Ctrl + O - открыть файл
  • Ctrl + S - сохранить файл
  • Ctrl + / - закомментировать/раскомментировать строку
  • Alt + Z - включить/выключить перенос строк

Теперь, когда вы знакомы с основами работы в VS Code, мы готовы перейти к изучению HTML - языка разметки, который является основой всех веб-страниц.
