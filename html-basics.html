Основы HTML

HTML (HyperText Markup Language) - это язык разметки, который используется для создания структуры веб-страницы. Он состоит из тегов, которые определяют различные элементы страницы.

Базовая структура HTML-документа

                    <!DOCTYPE html>
                    <html lang="ru">
                    <head>
                        <meta charset="UTF-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Название страницы</title>
                    </head>
                    <body>
                        <!-- Содержимое страницы -->
                    </body>
                    </html>

Основные HTML-теги

Тег Описание Пример
<h1>-<h6> Заголовки разного уровня (h1 - самый важный) <h1>Главный заголовок</h1>
<p> Абзац текста <p>Это абзац текста.</p>
<a> Ссылка <a href="https://example.com">Ссылка</a>
<img> Изображение <img src="image.jpg" alt="Описание">
<ul>, <ol>, <li> Списки (маркированный и нумерованный) <ul><li>Пункт 1</li><li>Пункт 2</li></ul>
<div> Блочный контейнер <div>Блок содержимого</div>
<span> Строчный контейнер <span>Часть текста</span>
<header> Шапка сайта или раздела <header>Шапка сайта</header>
<footer> Подвал сайта или раздела <footer>Подвал сайта</footer>
<nav> Навигационное меню <nav><a href="#">Ссылка 1</a></nav>
<section> Секция документа <section>Секция контента</section>
<article> Самостоятельный раздел контента <article>Статья</article>
<table>, <tr>, <td>, <th> Таблица, строка, ячейка, заголовок ячейки <table><tr><td>Ячейка</td></tr></table>
<form>, <input>, <button> Форма, поле ввода, кнопка <form><input type="text"><button>Отправить</button></form>

Атрибуты HTML-тегов

Теги могут иметь атрибуты, которые предоставляют дополнительную информацию:

  • class - определяет один или несколько классов для элемента (используется в CSS)
  • id - уникальный идентификатор элемента
  • style - встроенные CSS-стили
  • src - путь к файлу (для img, script и др.)
  • href - URL-адрес (для ссылок)
  • alt - альтернативный текст (для изображений)
  • width/height - ширина и высота элемента
<a href="https://example.com" class="link" id="main-link" style="color: blue;">Пример ссылки</a>

Это основные теги и атрибуты HTML, которые позволят вам создать структуру веб-страницы. В следующем разделе мы изучим CSS, который используется для оформления и стилизации HTML-элементов.
