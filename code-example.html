Примеры кода с пояснениями

В этом разделе мы рассмотрим практические примеры объединения HTML и CSS с подробными комментариями к каждой части кода.

Пример 1: Базовая структура HTML-документа с CSS


                <!DOCTYPE html> <!-- Объявление типа документа -->   
                <html lang="ru"> <!-- Корневой элемент с указанием языка -->              
                <head><!-- Метаданные документа -->
                
                <meta charset="UTF-8"> <!-- Кодировка символов -->
                
                
                
                
                <meta name="viewport" content="width=device-width, initial-scale=1.0">  
                <title>Мой сайт </title> <!-- Заголовок вкладки браузера -->
               <style><!-- Подключение CSS -->
                
                
                
                
                /* Сброс стандартных отступов и полей */
                * {
                    margin: 0;
                    padding: 0;
                    box-sizing: border-box; /* Учет padding и border в ширине элемента */
                }
            
                /* Стили для body */
                body {
                    font-family: 'Arial', sans-serif;
                    line-height: 1.6;
                    color: #333;
                    background-color: #f9f9f9;
                    padding: 20px;
                }
                
                /* Стили для шапки */
                header {
                    background-color: #000;
                    color: white;
                    padding: 1rem 0;
                    margin-bottom: 20px;
                }
                
                /* Стили для навигации */
                nav {
                    display: flex; /* Flex-контейнер */
                    justify-content: center; /* Выравнивание по центру */
                    gap: 20px; /* Расстояние между элементами */
                }
                
                nav a {
                    color: white;
                    text-decoration: none;
                    padding: 5px 10px;
                }
                
                nav a:hover {
                    text-decoration: underline;
                }
                </style>
                </head>
                <body>
                <header>
                <nav>
                    <a href="#">Главная</a>
                    <a href="#">О нас</a>
                    <a href="#">Контакты</a>
                </nav>
                </header>
    
                <main>
                <h1>Добро пожаловать на мой сайт!</h1>
                <p>Это пример базовой структуры HTML-документа с CSS-стилями.</p>
                </main>
                
                <footer>
                
                </footer>
                </body>
                </html>

Пример 2: Карточка товара

                 

            <div class="product-card">
            <img src="product.jpg" alt="Название товара" class="product-image">
            <div class="product-info">
            <h3 class="product-title">Название товара</h3>
            <p class="product-description">Краткое описание товара...</p>
            <div class="product-price">999 ₽</div>
            <button class="add-to-cart">В корзину</button>
            </div>
            </div>

            <style>
            /* Стили для карточки товара */
            .product-card {
                width: 300px;
                border: 1px solid #ddd;
                border-radius: 8px;
                overflow: hidden; /* Скрытие выходящего за границы контента */
                box-shadow: 0 2px 5px rgba(0,0,0,0.1);
                transition: transform 0.3s; /* Плавная анимация */
            }
    
            .product-card:hover {
                transform: translateY(-5px); /* Эффект поднятия при наведении */
            }
            
            .product-image {
                width: 100%;
                height: 200px;
                object-fit: cover; /* Масштабирование изображения с сохранением пропорций */
            }
            
            .product-info {
                padding: 15px;
            }
            
            .product-title {
                margin: 0 0 10px 0;
                font-size: 1.2rem;
            }
            
            .product-description {
                color: #666;
                margin-bottom: 15px;
            }
            
            .product-price {
                font-size: 1.4rem;
                font-weight: bold;
                margin-bottom: 15px;
                color: #e74c3c;
            }
            
            .add-to-cart {
                width: 100%;
                padding: 10px;
                background-color: #3498db;
                color: white;
                border: none;
                border-radius: 4px;
                cursor: pointer; /* Изменение курсора при наведении */
                transition: background-color 0.3s;
            }
            
            .add-to-cart:hover {
                background-color: #2980b9;
            }
            </style>
